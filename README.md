CitcomSVE is a finite element package for modeling terrestrial planetary viscoelastic deformation in response to tidal and surface loads. This repository is for Version 3.0 -- an actively developing version. For a more stable version 3.0 with user's guide, input files, and cookbook examples, go to branch CitcomSVE-3.0 in https://github.com/shjzhong/CitcomSVE

This project is still a work in progress, as always for scientific projects. Users from scientific community may contact Shijie Zhong for more information in using CitcomSVE at shijie.zhong@colorado.edu.
